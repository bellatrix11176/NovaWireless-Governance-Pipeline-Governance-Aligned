# NovaWireless-Governance-Pipeline-Governance-Aligned

Trust Signal Health governance pipeline applied to governance-aligned call center data. Demonstrates correct metric behavior when no proxy gaming pressure is present — establishing the baseline against which distorted environments are measured.

---

⚠️ Resource Notice
This pipeline is designed to be run iteratively on small datasets. Do not run against large datasets in a single pass. Process one month at a time to avoid memory and CPU overload. Adjust chunk sizes based on your available system resources.

---

## What This Is

This is the companion pipeline to [NovaWireless-Governance-Pipeline](https://github.com/bellatrix11176/novawireless-governance-pipeline), which documented Goodhart's Law dynamics under legacy FCR metrics. This repo runs the identical pipeline on data generated by the [NovaWireless-GoodhartLab](https://github.com/bellatrix11176/NovaWireless-GoodhartLab) governance-aligned call generator.

The question being answered: **what does a healthy measurement environment actually look like in the Trust Signal Health framework?**

## Key Results

| Metric | Value |
|--------|-------|
| Rows analyzed | 122,801 |
| Months | 12 (2025-01 through 2025-12) |
| Reps | 250 |
| Proxy resolution rate | 76.9% |
| True resolution rate | 63.8% |
| Resolution inflation | 13.1 pp (operationally explained — see paper) |
| Bandaid credit rate | **0.0%** |
| Trust score mean | 87.2 |
| SII (System Integrity Index) | **5.2 / 100** |
| VETO alerts | **0** |
| WATCH alerts | **0** |

The 13.1 pp resolution gap is not a governance failure. It is entirely attributable to `unresolvable_clean` and `activation_failed` scenarios — genuinely difficult call types the system handles honestly. Clean scenarios show near-zero or negative gaps.

## Governance Signals

All six Trust Signal Health signals as defined in Aulabaugh (2026b):

| Signal | Full Name | Result |
|--------|-----------|--------|
| DAR | Delayed Adverse Rate | 0.175 normalized — only non-zero signal |
| DRL | Downstream Remediation Load | 0.000 — no distributional drift |
| DOV | Durable Outcome Validation | 0.000 — no predictive validity decay |
| POR | Proxy Overfit Ratio | 0.000 — proxy not racing ahead of true outcomes |
| TER | Terminal Exit Rate (diagnostic) | 0.270 = baseline churn — no retention signal in proxy |
| **SII** | **System Integrity Index** | **5.2 / 100 — OK** |

SII thresholds: WATCH >= 30, VETO >= 60.

## Comparison with the Distorted Environment

| Metric | Distorted | Governance-Aligned |
|--------|-----------|-------------------|
| Proxy FCR | 89.4% | 76.9% |
| True resolution | 47.3% | 63.8% |
| Resolution gap | 42.1 pp | 13.1 pp |
| Bandaid credit rate | 71.1% | 0.0% |
| SII | VETO range | 5.2 (OK) |
| Gaming propensity x gap | r = +0.461 | ~0 |

A lower proxy FCR in the governance-aligned environment is not a worse outcome. It reflects honest handling of difficult calls rather than gaming-inflated counts.
